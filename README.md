# nthuion-frontend

Frontend of NTHU ION

## Resources

* [Backend Repo](https://github.com/nthuion/nthuion-backend)

## Prerequisites

* node
* npm

## Environment Setup

```bash
npm install # installs dependencies
```

## Run in Develop Mode

```bash
npm run dev
```

## Run in Production Mode

```bash
npm start
```

## Run Tests

```bash
npm test
```

## Run Lint

```bash
npm run lint
```
